# MY_SHOOTER - Unreal Engine 5.4 Lyra Shooter Project

This is an Unreal Engine 5.4 project based on the Lyra Starter Game, featuring custom shooter scenarios and gameplay mechanics.

## Project Information

- **Engine Version**: Unreal Engine 5.4
- **Base Framework**: Lyra Starter Game
- **Project Type**: C++ Game Project

## Features

- Custom shooter maps and scenarios
- Extensive character and weapon assets
- Enhanced gameplay abilities
- Multiple game modes
- Rich environment assets

## Large File Storage

This repository uses **Git LFS** (Large File Storage) to manage binary assets efficiently. 

### Cloning this Repository

To clone this repository, you need to have Git LFS installed:

```bash
# Install Git LFS (if not already installed)
git lfs install

# Clone the repository
git clone <repository-url>
cd MyShooterScenarios
```

## Development Setup

1. Ensure you have Unreal Engine 5.4 installed
2. Clone this repository with Git LFS
3. Right-click on `MY_SHOOTER.uproject` and select "Generate Visual Studio project files"
4. Open `MY_SHOOTER.sln` in Visual Studio
5. Build the solution
6. Launch the project from Unreal Engine Editor

## Assets Included

- Multiple character packs (Paragon characters, Fighter animations, etc.)
- Environmental assets (Abandoned Factory, Construction Site, Sci-Fi environments)
- VFX packs (Blood effects, Magic effects, Energy beams)
- Audio packs (Combat sounds, Environment sounds, Vocalizations)
- Animation sets (Close combat, Boxing attacks, Spear animations)
